User Guide

Getting Started

Step-by-Step Guide

  1. Start Your Workout: Press the Start button when you begin your first set
  2. Complete Your Set: Do your set of exercises
  3. Mark Set Complete: Press Set Complete when you finish your set
  4. Rest: Rest until the timer tells you to start your next set
  5. Repeat: Repeat steps 2-4 for each set

Understanding the Interface

Main Controls

  • Start Button: Begins your workout timer
  • Pause/Resume Button: Temporarily stops and restarts your workout
  • Reset Button: Clears all current workout data
  • Set Complete Button: Records when you finish a set
  • Sensor Toggle: Turns automatic set detection on/off

Using the Sensor Feature

How to Set Up the Sensor

For best results with automatic set detection:

  • Place your device on a stable, flat surface
  • Ensure good lighting in your workout area
  • Keep the device within 2-3 meters of your workout space
  • Make sure the sensor has a clear view of your movement area

Sensor Calibration

If the sensor isn't detecting your sets accurately:

  1. Make sure the sensor is enabled (green "Sensor: ON" button)
  2. Check that your device is properly positioned
  3. Ensure there's adequate lighting
  4. Try doing a test set to see if it detects properly

Reading Your Stats

Understanding the Numbers

  • Sets Completed: Total number of sets you've finished in this workout
  • Current Rest: Time elapsed since your last set completion
  • Best Set Time: Your fastest (shortest) rest period between sets
  • Total Rest Time: Cumulative rest time for your entire workout
  • Total Workout Time: Overall duration of your current workout session

Tips for Best Results

Workout Timing Tips

  • Be consistent with your rest periods for better tracking
  • Use the same device for your workouts to maintain data consistency
  • Don't clear your browser data if you want to keep your workout history
  • Experiment with manual vs sensor mode to see which works better for you

Troubleshooting Common Issues

  • Timer not starting: Refresh the page and try again
  • Sensor not detecting: Check device placement and lighting
  • Stats seem wrong: Reset and start a new workout
  • Page not loading properly: Clear browser cache and reload
