Report an Issue

Something not working right? Let us know so we can fix it!

How to Report Issues

Send detailed information about the problem to: haqmalebhaam2025@gmail.com

What Information to Include

Essential Details

  • What you were doing: Describe what you were trying to do when the problem occurred
  • What happened: Describe what actually happened
  • What you expected: Describe what you expected to happen instead

Technical Information

  • Device type: Phone, tablet, or computer
  • Browser and version: Chrome, Firefox, Safari, etc.
  • Operating system: Windows, macOS, iOS, Android
  • SmartGymTimer version: If you know it (usually shown in About)

Error Details

  • Error messages: Copy any error messages exactly as they appear
  • Screenshots: Include screenshots if possible
  • Steps to reproduce: List the exact steps that cause the issue

Common Issues and Solutions

Timer Not Working

Try this first: Refresh the page and try again. If that doesn't work, clear your browser cache.

Sensor Not Detecting Sets

Check these things:

  • Is the sensor enabled (showing "Sensor: ON")?
  • Is your device on a stable surface?
  • Is there adequate lighting?
  • Is your device within 2-3 meters of your workout area?

Data Not Saving

Possible causes:

  • You might have cleared your browser data
  • Your browser might be blocking local storage
  • You might be using private/incognito mode

Before Reporting

Try these troubleshooting steps first:

  1. Refresh the page: Many issues are fixed with a simple refresh
  2. Clear browser cache: Old cached files can sometimes cause problems
  3. Try a different browser: See if the issue occurs in other browsers
  4. Check your internet connection: Some features might require internet access
  5. Restart your device: Sometimes a fresh start helps

What Happens After You Report an Issue

  • We'll acknowledge receipt of your report within 24 hours
  • We'll investigate the issue and try to reproduce it
  • If we can fix it quickly, we'll deploy a fix as soon as possible
  • For more complex issues, we'll add it to our development queue
  • We'll keep you updated on our progress if you provide contact information

Urgent Issues

If SmartGymTimer is completely unusable for you, please include "URGENT" in your subject line and we'll prioritize your issue.
